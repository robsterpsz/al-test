    public toMoment(date: Date, timezone: String): Object source 

    import {toMoment} from 'altest/src/dateUtils.js'

    Convert a Javascript Date into a Moment.js moment obj with the appropriate timezone. Using the returned moment, you can call for example 'moment.calendar()' to get a human readable relative time such as 'last Friday at 3:52 PM'.

    Params:
